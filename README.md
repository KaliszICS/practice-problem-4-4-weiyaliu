# Instructions  

1. Create a function called _hasCapital_() that take in a String.  return true if the String has a capital letter in it.  Return false otherwise.

2. Create a function called _isPrime_() that takes in an integer.  Using a for loop, determine whether the integer variable is prime or not.  If the number is prime, return true.  If it's not, return false.  Keep in mind that 1 is NOT a prime number.
